# Bebridge DeFi IR

스테이블코인 기반 DeFi 예치 플랫폼 **Bebridge**의 IR(Investor Relations) 프레젠테이션 웹사이트입니다.

## 핵심 논리

| 구분 | 내용 |
|------|------|
| **시장변화** | 스테이블코인 제도권 진입 + 금융기관 수용 시작 |
| **기회** | DeFi 효율성은 저리스크 상품(예적금)에서 극대화 — 수익률 +100% |
| **한국 시장** | 예적금 비중 높고, 달러 선호 급증하는 최적 시장 |
| **Problem** | 일반인은 인식·사용성 장벽으로 DeFi 이용 불가 |
| **Solution** | "달러 예금" 포지셔닝 + DeFi 예치 전용 지갑 앱 |
| **BM** | SOM 5,300억 → SAM 20조 → TAM 2,200조원 |
| **Vision** | 한국 #1 달러 핀테크 → 글로벌 핀테크 유니콘 |
